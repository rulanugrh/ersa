<h3 align="center"> Ersa </h3>
<h6 align="center"> Lookup DNS via command line </h6>

**install from script**
```
git clone https://github.com/rulanugrh/ersa.git
cd ersa
sudo ./install.sh
```

**self build**
```
git clone https://github.com/rulanugrh/ersa.git
cd ersa
go build -o ersa
```

**install (windows user)**
```
you can copy file `build/ersa.exe` if you are arch x64 bit
```

**how to use**
```
ersa [DOMAIN_NAME]
```
